# DTC Competitor Intelligence Framework

> 一套能直接落地的 DTC 独立站竞对深度拆解方法论——从流量底盘摸底到90天行动计划，每步都有具体 SOP。

## 一句话

拆竞对不是为了写报告，是为了**找到对手的弱势 + 可抄的做法 + 落成自己的行动计划**。

## 框架结构

```
竞调三问 (锚定方向)
    ↓
Phase 1:  流量底盘摸底 — 设备/新回访/质量/地域
Phase 2:  受众画像与决策链路 — 谁买、在哪、怎么买单
Phase 3:  渠道构成与质量审计 — 六渠道拆解 + 强弱项判断
Phase 4:  SEO与内容策略深拆 — 关键词/内容集群/外链
Phase 5:  广告打法逆向工程 — 素材/人群/落地页反推
Phase 6:  转化漏斗与GMV测算 — 三角验证法交叉定价
Phase 7:  技术栈扫描 — 建站/插件/支付/物流
Phase 8:  品牌口碑与用户洞察 — Trustpilot/Reddit/评论区
Phase 9:  SWOT聚合 — 抄什么、打什么、建什么壁垒
Phase 10: 90天行动计划 — 谁做、怎么做、做到什么程度、ROI
    ↓
附录A-C: 执行SOP速查 (产品上线 → 站外SEO → 全栈架构)
```

## SKILL.md 是什么

[SKILL.md](./SKILL.md) 是 Claude Code 可直接加载的 AI skill。在 Claude Code 中输入 `/竞对调研 https://xxx.com` 即可自动按此框架执行全流程竞调。

如果你是：
- **出海 DTC 创业者** → 直接用 SKILL.md 做方法论参考
- **AI Agent 开发者** → 参考 YAML frontmatter 结构，了解如何封装领域知识为 skill
- **跨境运营** → 看 examples 了解完整竞调报告产出标准
