# Phase 7: 技术与基础设施扫描

## 检查清单

这一层直接影响转化率和运营效率，但多数竞调会跳过去。

### 建站平台
```
□ CMS/建站平台: Shopify / WooCommerce / Magento / 自建 / 其他
  → 检测方法: BuiltWith / Wappalyzer / 页面源码特征
  → Shopify特征: /cdn.shopify.com/ 路径, myshopify.com CNAME
  → WooCommerce特征: /wp-content/plugins/woocommerce/
```

### 插件生态
```
□ 评论插件: Judge.me / Yotpo / Loox / Stamped.io
□ 弹窗/邮件捕获: Klaviyo / Privy / Justuno / OptinMonster
□ 推荐/交叉销售: LimeSpot / Wiser / ReConvert / Also Bought
□ 弃购挽回: Klaviyo / Mailchimp / SendGrid / SMSBump
```

### 支付方式
```
□ PayPal / Stripe / Shopify Payments
□ BNPL: Klarna / Afterpay / Affirm
□ 本地支付: iDEAL / Sofort / Giropay（欧洲站常见）
```

### 物流/履约
```
□ 运费策略: 免邮门槛 / 固定运费 / 阶梯运费
□ 配送时效承诺
□ 退货政策: 退货窗口 / 谁付运费 / restocking fee
□ 仓储线索: 多仓分布 / 海外仓 / 自有仓
```

### 网站速度
```
□ GTmetrix / PageSpeed Insights 评分
□ 速度差距 = 直接转化差距（电商每慢1秒转化降7%）
```

### 多语言/多市场
```
□ 是否有 /fr/ /de/ /ko/ 等语言路径
□ Amazon店铺分布哪些国家
□ 是否有B2B独立系统（如 .net 域名）
```
