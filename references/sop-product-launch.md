# 附录A: 产品上线全流程SOP

> 竞调结束后，如果决定"抄"竞对的产品策略，用此SOP执行站内落地。

## A1. 关键词研究 (约1h)

```
□ 挖掘核心关键词 + 长尾关键词
  工具: Google Keyword Planner / SEMrush
□ 分析竞对关键词覆盖（结合Phase 4竞调发现）
□ 匹配用户搜索意图
□ 确定产品英文名称: 简洁易记 + 包含核心关键词
□ 撰写产品描述: 核心卖点 / 材质 / 尺寸 / 解决痛点 / 使用场景 → 全部融入SEO关键词
□ 查找SKU
□ 确认GTIN码
```

## A2. 产品发布 (20min-3天，含素材制作)

```
□ 产品定价: 平衡利润率与竞争力
  - 参考Phase 3竞对定价 + Phase 6 GMV反推的定价空间
□ 库存管理设置
□ 写产品详情图脚本 (基于产品描述)
□ 详情图 → 对接摄影师/设计师 (1-3天)
□ 产品视频脚本 → 对接摄影师 (1-3天)
□ 每张图片添加Alt标签: 含关键词 + 描述图片内容
□ 发布产品
```
> 工具: ERP / CMS后台 / GPT4o / 图片压缩工具

## A3. 产品页面 & SEO优化 (约2-3h/页)

```
□ SurferSEO Audit + Content分析 → 收集需补充的关键词
□ 关键词与前期产品描述做匹配 (GPT辅助)
□ 整合生成最终产品描述，补齐详情图脚本
□ 二次Audit: 收集关键词 / 客户分群 / 痛点 / 常见问题
□ 对关键词分类 → 创建FAQ内容
□ HTML或网站组件实现FAQ
□ 按详情图顺序上传对应产品描述
□ 检测Audit最终分数
□ 优化 Meta标题 / Meta描述 / URL
□ 上传 Meta 数据
□ 检查所有图片 ALT
□ H1 / H2 / H3 标签层级检查
□ 设置相关产品推荐 (cross-sell / upsell)
□ 收集好评截图 → 上传 REVIEW
```

## A4. 移动端优化

```
□ 移动端友好性测试 (Google Mobile-Friendly Test)
□ 页面加载速度优化 (PageSpeed Insights)
```

## 工具速查

| 步骤 | 工具 |
|------|------|
| 关键词研究 | Google Keyword Planner / SEMrush |
| ERP管理 | ERP / GTIN码查询 (gds.org.cn) |
| 内容生成 | GPT4o |
| 素材处理 | CMS后台 / 图片压缩工具 |
| SEO审计 | SurferSEO |
| 速度测试 | PageSpeed Insights / Google Mobile-Friendly Test |
