## 优先级定义

- **P0（立即，0-30天）**: 照抄竞对已验证的高ROI动作，不需要创新的
- **P1（30-60天）**: 在竞对弱项上建立差异化优势
- **P2（60-90天）**: 建竞对抄不走的长期壁垒
- **P3（90天+）**: 需更多前置条件，暂缓但不忽略

## 每行必须回答

```
这个动作 → 预期带来多少流量/转化/GMV?
如果做不到 → 风险/止损线在哪？
```

## 资源总览

用表格输出初期团队配置和月预算。

## 风险与止损线

每个核心风险标注：触发条件 → 止损动作

## 注意事项

- 行动计划不是研究报告的附录，而是调研的核心交付物
- 每个动作必须有"竞对依据"——不能凭空想
- 如果某些动作需要验证（如广告ROI不确定），先小预算测试再放量
- 用户如果提供了自己站背景，调整优先级和资源估算
