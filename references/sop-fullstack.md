# 附录C: 独立站全栈架构速查

## C1. 渠道扩展全景

竞调完成后，规划自己的渠道矩阵：

```
可扩展渠道清单:
  □ 社媒广告 (FB/IG/TT)
  □ 联盟营销 (Affiliate)
  □ 红人营销 (KOL/KOC)
  □ 社区运营 (FB Groups / Discord / Reddit)
  □ 区域渠道 (OZON / Yandex / 本地电商)
```

## C2. 网站架构蓝图

### Menu导航结构
```
□ 按品类 (By Category)
□ 按场景 (By Use Case)
□ Sale / 促销区
□ Search (CMS站内搜索)
□ 产品分类 A-Z
□ 特色产品推荐
□ 促销活动页
□ 购买 & 订单信息
□ 服务条款 (隐私 / 退货 / 保修)
□ FAQ
□ 客服 & 联系方式
□ 用户评价汇总
```

### 首页 (Home Page) 必备模块
```
□ Hero Banner (主打产品/促销)
□ 产品分类入口 (Collection Cards)
□ 特色产品 (Featured Products)
□ 用户评价 (Social Proof)
□ 促销活动 (Promo Section)
□ Blog入口
□ Footer: About Us / Contact / Policy / FAQ
```

### 产品页 (PDP) 必备模块
```
□ 产品主图 + 多角度图 + 细节图
□ 产品标题（含核心关键词）
□ 价格 + 折扣标签
□ 变体选择（颜色/尺寸/配置）
□ Add to Cart CTA
□ 产品描述（SEO优化）
□ 技术规格表
□ FAQ（结构化数据）
□ 用户评价（带图）
□ Frequently Bought Together / 交叉推荐
□ 运费 & 退换政策摘要
```

## C3. 买量落地体系

### 搜索渠道
```
□ Google Ads SOP
  - 关键词研究 → 品类词/竞品词/品牌词分组
  - 产品定向 (Product Listing Ads / Google Shopping)
  - Display & Video
  - Remarketing (弃购 → 浏览 → 搜索重定向)
  - 预算: 小额测试起, ROAS≥3后放量
```

### KOL/内容渠道
```
□ KOL/KOC 合作SOP
  - 影响者筛选: 粉丝量 / 互动率 / 受众匹配 / 历史合作品牌
  - 合作模式: 纯样品置换 / 固定费用 / CPS分佣
  - 内容分发: YouTube详细评测 / IG Reels开箱 / TikTok短视频

□ YouTube SOP
  - 产品开箱/安装教程/对比评测/长期使用报告
  - 每视频配套: SEO标题+描述关键词+独立站落地页链接
  - 自有频道 + KOL频道双轨
```
