# Phase 4: SEO与内容策略深拆

## 搜索策略

```
并行搜索:
1. "[域名] SEO organic keywords ranking semrush ahrefs"
2. "[域名] backlinks referring domains domain rating"
3. "site:[域名] blog articles content"
4. "site:[域名] products categories collection"
```

## 四维拆解

### 关键词覆盖
- 品牌词 vs 品类词 vs 信息词 vs 长尾词 占比
- 工具: SEMrush / Ahrefs Organic Research（免费版也可看大概）
- 输出判断: 品牌驱动还是品类驱动？

### 页面类型构成
- 产品页 vs 博客 vs 品类页 vs 工具页 vs 关于我们
- 哪个类型的页面贡献最多SEO流量？
- 判断: 内容驱动型还是商品驱动型？

### 内容集群策略
- 是否有Topic Cluster结构？Pillar Page → Cluster Content → Internal Linking
- 是否有内容护城河？（博客量 > 50篇且有持续更新 = 形成护城河）
- 文章类型分布：品类教育 / How-to / 买家指南 / 对比评测 / 新闻

### 外链画像
- Domain Rating / DR（估算）
- Referring Domains 增长曲线
- 外链类型: 媒体评测 / PR / 社区 / 垃圾外链 / 合作伙伴
- 哪些外链渠道我们也能做？

## 可抄的SEO策略清单

```
1. 他们排名前10的非品牌词中，哪些我们也可以做？
2. 他们哪些高转化页面我们缺少？
3. 他们的内容集群结构我们能直接复用吗？
4. 他们获得外链的渠道（媒体评测/行业论坛/合作伙伴）我们能对接吗？
```

## SEO能力评级

- **成熟**: DR>50, 博客>100篇, 品类词排名靠前, 明显的内容集群结构
- **中等**: DR 20-50, 有博客但量不大, 品牌词排名好但品类词一般
- **初级**: DR<20, 博客极少, 仅品牌词有排名, 无明显内容策略
- **未做**: 基本没有SEO痕迹
