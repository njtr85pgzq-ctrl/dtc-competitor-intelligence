# Phase 7: 定价情报深拆

## 目标

搞清楚竞对的价格体系：锚点定价、折扣策略、运费策略、捆绑策略。回答"我在什么价位打他们最有优势？"

## 搜索策略

```
并行搜索:
1. "[品牌名] pricing discount coupon code promotion"
2. "[品牌名] shipping cost free shipping policy"
3. "[品类] price comparison best price 2026"
4. "[品牌名] bundle deal package offer"
```

## 必须输出

1. **价格定位一句话**: "竞对在这个品类是____价位段（低/中/高），核心价格区间 $____ - $____"
2. **我们的定价策略**: "我们的价格应该是竞对的 ____%（高5%打品牌 / 持平 / 低5%打性价比）"
3. **3个可立即用的价格钩子**: 基于差评数据，设计价格相关的营销钩子
