# Phase 2: 受众画像与决策链路

## 受众数据源

按优先级使用以下数据源：

| 优先级 | 数据源 | 能拿到的信息 | 工具 |
|--------|-------|-------------|------|
| 1 | 流量平台 | 年龄/性别/兴趣分类 | SimilarWeb Audience |
| 2 | 社媒互动 | 粉丝画像/活跃时段/内容偏好 | FB Audience Insights / IG |
| 3 | 搜索行为 | 搜索意图/关联词 | Google Trends / SEMrush |
| 4 | 评论/社区 | 真实用户语言/痛点/场景 | Reddit / Trustpilot / YouTube评论区 |
| 5 | 广告受众 | 投放定向 → 反推目标人群 | FB Ads Library + AdTargeting |

## 搜索策略

```
并行搜索:
1. "[品牌名] audience demographics age gender"
2. "[品牌名] customer profile target market"
3. "[品类] Reddit community discussion user profile"
4. "[品牌名] social media followers Instagram Facebook"
```
